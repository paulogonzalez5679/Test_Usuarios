En cuanto al front-end, acceder a la carpeta front end, ejecutar el comando npm i --force, y esperar que se culmine la instalacion de los modulos, una vez finalizado, ejecutar ionic s --lab, para tener una visualizacion en un emulador de un dispositivo m√≥vil.
